The Open Digital Archaeology Textbook

An introduction to the issues, methods, and techniques of digital archaeology, integrated with working code and virtual computing environments or 'notebooks' written in Python or R. The only thing you'll need is a browser! Intended for second or third year students, all materials are creative-commons licensed, and may be remixed to suit your instructional purposes. Funded by eCampusOntario.

Underlying Technologies

Bookdown

We wrote ODATE using Bookdown, a system for writing in R Markdown (plain text but interwoven with the R statistical computing language) and then automatically transforming the text into HTML, PDF, and Epub. Each section of the book is available in a plaintext file. Reorganize or remix with other materials in a folder, and Bookdown will create a new book. Bookdown is used within R Studio.

Find Out More about Bookdown

Jupyter

'The Jupyter Notebook is an open-source web application that allows you to create and share documents that contain live code, equations, visualizations and narrative text. Uses include: data cleaning and transformation, numerical simulation, statistical modeling, data visualization, machine learning, and much more.'
